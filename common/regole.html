Regole Punti:
- chi vince più game vince la giornata
- non si può iniziare un game se mancano meno di 25 minuti alla fine della prenotazione
- la giornata può finire in pareggio
- vince il game chi arriva prima a 6 set vinti (o 7 se si arriva a 5 set pari)
- nel 7' set i punti sono 15/30/40/eventuli vantaggi
